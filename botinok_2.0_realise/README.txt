1) Для запуска бота введите команду: /start

2) После сообщения о приветствии, сразу смените пароль Администратора командой (ВМЕСТО new_password введите ВАШ новый пароль): /admin 123 /change_admin_password new_password

3) После сообщения о приветствии, введите команду /admin your_password /add_channel your_tg_channel
ВМЕСТО your_tg_channel введите ссылку на нужный телеграмм-канал БЕЗ https://t.me/ и @, в формате: botinok_test

4) Повторите действие 2) для добавления нужного количества каналов.

5) Назначьте бота в нужных телеграмм-каналах АДМИНИСТРАТОРОМ! и снимите галочку с пункта "Отправка сообщений" или с похожего по смыслу значения.

6) Бот готов к использованию!

СПИСОК ВСЕХ КОМАНД АДМИНА:

ВМЕСТО your_password вводите свой пароль от Администратора!

ВМЕСТО your_tg_channel вводите ссылку на телеграмм-канал БЕЗ https://t.me/ и @, в формате: botinok_test

ВМЕСТО your_chatGPT_token вводите ВАШ новый токен chatGPT


1) /admin your_password /change_admin_password new_password - меняет старый пароль Администратора на новый.

2) /admin your_password /add_channel your_tg_channel - добавляет телеграмм-канал.

3) /admin your_password /delete_channel your_tg_channel - удаляет телеграмм-канал.

4) /admin your_password /change_gpt_key your_chatGPT_token - меняет токен chatGPT на новый.
